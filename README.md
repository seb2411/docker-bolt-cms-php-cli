DOCKER IMAGE FOR BOLT CMS CLI
=============================

A docker image to run [Bolt CMS](https://www.bolt.cm) commands based on Alpine Linux and Php CLI 7.1.

The image come with the following php libraries:

- Curl
- Gd
- Intl
- Xml
- Pdo Mysql
- mbstring
- opcache
- posix
- fileinfo
- exif
- zip
